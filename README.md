# isaachulse.github.io

## Running locally...

### Prerequisites

1. Ensure you have Ruby 2.1.0 or higher installed:

``` shell
$ ruby --version
```

2. Ensure bundler is installed:

``` shell
$ gem install bundler
```

3. Install Jekyll and other dependencies from the GitHub Pages gem: 

``` shell
$ bundle install
```

### To serve

1. Run Jekyll Serve

``` shell
bundle exec jekyll serve
```

2. Open in browser

```
http://localhost:4000
```
